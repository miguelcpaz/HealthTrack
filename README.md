# 🏥 HealthTrack

O **HealthTrack** é um sistema web para **gerenciamento e acompanhamento de pacientes**, desenvolvido com foco em organização clínica, controle de informações e boas práticas de desenvolvimento full stack.

O projeto simula um ambiente hospitalar ou ambulatorial, permitindo que profissionais da saúde cadastrem, acompanhem e gerenciem pacientes de forma segura e estruturada.

## 🎯 Objetivo do Projeto

O HealthTrack foi desenvolvido com fins **educacionais**, com o objetivo de:

- Desenvolver uma **API REST completa**
- Trabalhar com **Node.js e PostgreSQL**
- Aplicar **autenticação e controle de acesso**
- Integrar **front-end e back-end**
- Simular um **sistema real da área da saúde**

---

## 🧩 Funcionalidades

- 🔐 Sistema de **login e autenticação**
- 👥 Controle de acesso por usuário
- 📋 Cadastro de pacientes
- 🧑‍⚕️ Listagem e visualização de pacientes
- 📝 Registro de relatórios clínicos
- 💊 Controle de prescrições
- 🏨 Acompanhamento de estadia/internação
- ✏️ Edição e exclusão de registros
